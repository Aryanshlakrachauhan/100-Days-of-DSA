### Double Ended Queue (Deque)

Problem Statement:
Implement a Double Ended Queue (Deque) that supports insertion and deletion from both ends.

Supported Operations:
- pushFront x
- pushRear x
- popFront
- popRear
- peekFront
- peekRear

Input Format:
- First line contains integer N (number of operations)
- Next N lines contain one operation per line

Output Format:
- Print the result of pop and peek operations
- Print -1 if operation cannot be performed

Example:
Input:
6
pushFront 10
pushRear 20
peekFront
popRear
peekRear
popFront

Output:
10
20
10
