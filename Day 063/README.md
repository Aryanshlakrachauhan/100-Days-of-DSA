### Depth First Search (DFS)

Problem Statement:
Perform Depth First Search traversal starting from a given source vertex.

Input Format:
- First line contains n m
- Next m lines contain edges
- Last line contains source vertex

Output Format:
- Print DFS traversal order

Example:
Input:
5 4
1 2
1 3
2 4
3 5
1

Output:
1 2 4 3 5

Explanation:
DFS explores as far as possible before backtracking.
