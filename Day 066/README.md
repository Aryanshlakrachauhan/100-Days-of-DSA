### Cycle Detection in Directed Graph

Problem Statement:
Detect whether a directed graph contains a cycle.

Input Format:
- First line contains n m
- Next m lines contain directed edges

Output Format:
- Print YES if cycle exists, otherwise NO

Example:
Input:
3 3
1 2
2 3
3 1

Output:
YES

Explanation:
A back-edge in DFS recursion stack confirms a cycle.
