### Topological Sort (Kahn’s BFS Algorithm)

Problem Statement:
Perform Topological Sort of a Directed Acyclic Graph using Kahn’s BFS algorithm.

Input Format:
- First line contains n m
- Next m lines contain directed edges

Output Format:
- Print topological order

Example:
Input:
6 6
5 2
5 0
4 0
4 1
2 3
3 1

Output:
4 5 2 0 3 1

Explanation:
Nodes with zero in-degree are processed first using a queue.
