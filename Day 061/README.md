### Graph Representation Using Adjacency Matrix

Problem Statement:
Represent a graph using an adjacency matrix.

Input Format:
- First line contains n m
- Next m lines contain edges u v

Output Format:
- Print adjacency matrix

Example:
Input:
4 3
1 2
2 3
3 4

Output:
0 1 0 0
1 0 1 0
0 1 0 1
0 0 1 0

Explanation:
Matrix cell (i,j) is 1 if edge exists between i and j.
