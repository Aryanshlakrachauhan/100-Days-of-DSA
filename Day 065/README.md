### Cycle Detection in Undirected Graph (DFS)

Problem Statement:
Detect whether an undirected graph contains a cycle using DFS.

Input Format:
- First line contains n m
- Next m lines contain edges

Output Format:
- Print YES if cycle exists, otherwise NO

Example:
Input:
4 4
1 2
2 3
3 4
4 2

Output:
YES

Explanation:
Back-edge during DFS indicates a cycle.
