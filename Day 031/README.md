### Implement Stack Using Array

Problem: Implement Stack Using Array - Implement using linked list with dynamic memory allocation.

Input:
- First line: integer n
- Second line: n space-separated integers

Output:
- Print stack elements from top to bottom, space-separated

Example:
Input:
5
10 20 30 40 50

Output:
50 40 30 20 10

Explanation:
Use array and top pointer. Push inserts at top, pop removes from top. Print elements from top to bottom.
