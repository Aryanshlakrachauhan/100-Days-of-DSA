## Question 1

### Check Identical Binary Trees

Problem Statement:
Given two binary trees, determine whether they are identical. Two binary trees are considered identical if they have the same structure and the same node values.

Input Format:
- First line contains integer N (number of nodes in first tree)
- Second line contains level-order traversal of first tree (-1 represents NULL)
- Third line contains integer M (number of nodes in second tree)
- Fourth line contains level-order traversal of second tree (-1 represents NULL)

Output Format:
- Print "YES" if both trees are identical, otherwise print "NO".

Example:
Input:
7
1 2 3 4 -1 -1 -1
7
1 2 3 4 -1 -1 -1

Output:
YES
