### Topological Sort (DFS)

Problem Statement:
Perform Topological Sort of a Directed Acyclic Graph using DFS.

Input Format:
- First line contains n m
- Next m lines contain directed edges

Output Format:
- Print topological order

Example:
Input:
6 6
5 2
5 0
4 0
4 1
2 3
3 1

Output:
5 4 2 3 1 0

Explanation:
Vertices are ordered such that for every edge u → v, u appears before v.
