### Graph Representation Using Adjacency List

Problem Statement:
Represent a graph using adjacency list.

Input Format:
- First line contains n m
- Next m lines contain edges u v

Output Format:
- Print adjacency list

Example:
Input:
4 3
1 2
1 3
2 4

Output:
1 -> 2 3
2 -> 4
3 ->
4 ->

Explanation:
Each vertex stores a list of its adjacent vertices.
