### Implement Stack Using Linked List

Problem: Implement Stack Using Linked List - Implement using linked list with dynamic memory allocation.

Input:
- First line: integer n
- Second line: n space-separated integers

Output:
- Print stack elements from top to bottom, space-separated

Example:
Input:
5
10 20 30 40 50

Output:
50 40 30 20 10

Explanation:
Push inserts at head, pop removes head. Traverse linked list from head to print stack.
