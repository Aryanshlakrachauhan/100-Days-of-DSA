### Delete Node from BST

Problem Statement:
Delete a given node from a Binary Search Tree while maintaining BST properties.

Input Format:
- First line contains integer N
- Second line contains N space-separated integers
- Third line contains key to delete

Output Format:
- Print inorder traversal after deletion

Example:
Input:
5
4 2 6 1 3
2

Output:
1 3 4 6
